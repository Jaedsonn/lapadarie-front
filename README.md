<h2 id="started">Vamos rodar o projeto?</h2>

<h3>Prerequesitos</h3>

- [NodeJS](https://github.com/)
- [Git 2](https://github.com)

<h3>Clonando</h3>


```bash
git clone https://github.com/Jaedsonn/lapadarie-front.git
```

<h3>Iniciando o projeto</h3>

How to start your project

```bash
cd lapadarie-front
npm i
npm run dev
```
